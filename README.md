# MeetKey
> 회의 내용을 분석하여 부분 업무 자동화를 할 수 있게 돕는 앱

## 이런 기능이 있습니다!
* 회원 관리
* 회의 스크립트 제공
* 전체 회의 요약
* 발언자별 회의 분석

## 이런 기술들을 썼어요!
* Naver CSR
* MVC
* Volley
